About Andrew

Additionally, I was on the tennis team and took courses at Baypath College through the high school's dual enrollment program. Good Software

I run GNU-Linux, and am interested in free / open source software. I'll post my recommendations here.
  • Linux Distro: I currently run Fedora. In the past I've ran Ubuntu and Arch Linux. I'd still be running Arch, but it turns having a computer into a full time hobby.
  • Productivity tools: I've recently discovered Org Mode, a plugin for emacs (it probably came with your emacs installation). Org Mode stores your data (notes, to-do lists, anything really) in easy-to-format text files. It supports collapsible views and exports to ASCII, HTML, and LaTeX. It also has LaTeX support, which is a big plus. Essentially, it makes not-taking on a computer possible for mathematicians.
  • Mathematical Computing: Sage is awesome. It's built on Python, and supports many mathematical programming languages, too (GAP and GP/Pari, to name two). The notebook interface is good, and you can also test out sage on sage notebook or the sage project.
  • General Computing: I've taken a liking to Python. It's definitely faster to code in than something like Java. I'll update this after I do some more coding in Python.
